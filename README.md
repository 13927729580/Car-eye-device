Car-eye-device
 
car-eye 设备子系统提供了一整套硬件设计，固件程序，系统软件，行车记录仪等功能模块得解决方案

其中SH-CAMERA 是设备端行车记录仪模块，该功能模块实现了传统的DVR功能。还支持远程控制，支持远程观看实时视频和检索录像。


多路行车记录仪：sh_camera
